环境配置

我的环境配置：

环境： Vmware workstation 模拟
OS版本：Ubuntu 16.10 (GNU/Linux 4.8.0-22-generic x86_64)
基础软件： git open-sshd python
ip地址： 192.168.172.128

安装步骤

  1. 创建stack用户
    $ adduser stack
    … type password …

    $ echo “stack ALL=(ALL) NOPASSWD: ALL” >> /etc/sudoers
    $ su stack
    然后设置stack的用户目录为 /opt/stack

  2. 拷贝devstack
    在/opt/stack目录中执行：
    git clone https://github.com/openstack-dev/devstack.git -b stable/ocata

  3. 配置local.conf文件
    cp /opt/stack/devstack/samples/local.conf .

    然后对其做编辑（原来在的就不写出来了）：

[[local|localrc]]
\# 相关密码配置
ADMIN_PASSWORD=lrk123
DATABASE_PASSWORD=lrk123
RABBIT_PASSWORD=lrk123
SERVICE_PASSWORD=$ADMIN_PASSWORD

\# 主机地址
HOST_IP=192.168.172.128

\# 使用国内较快的git仓库下载openstack仓库
GIT_BASE=http://git.trystack.cn
NOVNC_REPO=http://git.trystack.cn/kanaka/noVNC.git
SPICE_REPO=http://git.trystack.cn/git/spice/spice-html5.git

\# 制定组件的git分支
KEYSTONE_BRANCH=stable/ocata
NOVA_BRANCH=stable/ocata
NEUTRON_BRANCH=stable/ocata
SWIFT_BRANCH=stable/ocata
GLANCE_BRANCH=stable/ocata
CINDER_BRANCH=stable/ocata

  1. 配置pip源
    devstack默认python源为https://pypi.python.org。可以从https://www.pypi-mirrors.org/选一个国内的镜像，把镜像地址写到pip.conf文件，并放到/root/.pip/pip.conf以及/opt/stack/.pip/pip.conf

    内容为：

[global]
index-url = https://pypi.douban.com/simple
  1. 执行命令：./stack.sh

  2. 中途遇到个问题，中断了执行，屏幕报了这么个错误：
    No hosts found to map to cell, exiting
    查看 http://stackoverflow.com/questions/41064739/openstack-no-hosts-found-to-map-to-cell-exiting，后，完美解决。

    问题就在与，需要把nova-cell组件也使能起来：

[[local|localrc]]
**enable_service n-cell**
\# 相关密码配置
...

    按照指示登陆：http://192.168.172.128/dashboard/
